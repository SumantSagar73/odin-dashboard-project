# Admin Dashboard

## Description
This project is an admin dashboard built using HTML, CSS (Grid), and JavaScript. The layout includes a sidebar, header, main content area, and a right sidebar with sections for projects, announcements, and trending items. The dashboard also features a dark mode toggle and a collapsible sidebar.

## Features
- Responsive grid-based layout using CSS Grid
- Sidebar with navigation links
- Header with search bar, notifications, and user profile
- Main content area displaying projects in a grid format
- Right sidebar containing announcements and trending users
- Dark mode toggle with local storage support
- Collapsible sidebar

## Technologies Used
- HTML
- CSS (CSS Grid)
- JavaScript (for interactivity)
- Google Material Icons
- Google Fonts (Roboto)

## Installation
1. Clone the repository:
   ```sh
   git clone https://github.com/SumantSagar73/odin-dashboard-project.git
   ```
2. Navigate to the project directory:
   ```sh
   cd odin-dashboard-project
   ```
3. Open `index.html` in your browser.

## Usage
- Click the sidebar toggle button to collapse/expand the sidebar.
- Use the search bar to simulate searching functionality.
- Click the dark mode toggle to switch themes (preference is saved in local storage).
